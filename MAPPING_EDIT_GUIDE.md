# מדריך עריכת מיפויים

## איך לערוך מיפוי קיים?

### תרחיש: שינוי קטגוריה של פריט

נניח שיש לך מיפוי:
```
"רמי לוי" → "כלכלה"
```

ואתה רוצה לשנות אותו ל:
```
"רמי לוי" → "מזון"
```

### שלבים:

#### 1. עבור לטאב "🗺️ מיפוי קטגוריות"

#### 2. מצא את המיפוי בטבלה
```
┌─────────────┬──────────────┬──────────────┐
│ פריט        │ קטגוריה     │ פעולות       │
├─────────────┼──────────────┼──────────────┤
│ רמי לוי     │ כלכלה        │ ✏️  🗑️      │
└─────────────┴──────────────┴──────────────┘
```

#### 3. לחץ על כפתור העריכה (✏️)
טופס עריכה יופיע:
```
┌─────────────────────────────────┐
│ עריכת מיפוי                     │
│                                 │
│ פריט: רמי לוי (לא ניתן לשינוי) │
│ קטגוריה: [בחר מהרשימה ▼]       │
│                                 │
│ [💾 עדכן]  [❌ בטל]            │
└─────────────────────────────────┘
```

#### 4. בחר קטגוריה חדשה
- הפריט "רמי לוי" מוצג אך לא ניתן לשינוי (זה נועל כדי לשמור על עקביות)
- בחר קטגוריה חדשה מהרשימה הנפתחת
- לחץ "💾 עדכן"

#### 5. ✅ המיפוי עודכן!
הודעת הצלחה תופיע, והטבלה תתעדכן אוטומטית.

---

## שאלות נפוצות

### ש: למה לא ניתן לשנות את שם הפריט?
**ת:** כדי לשמור על עקביות. אם רוצים פריט אחר, עדיף להוסיף מיפוי חדש.

### ש: מה קורה לעסקאות קיימות אחרי שינוי מיפוי?
**ת:** העסקאות לא משתנות אוטומטית. הן שומרות על הקטגוריה שהייתה להן בזמן היצירה. המיפוי החדש יחול רק על עסקאות חדשות.

### ש: איך אני מעדכן גם את העסקאות הישנות?
**ת:** צריך לערוך כל עסקה ידנית, או למחוק ולהוסיף מחדש.

### ש: אפשר לשנות גם את שם הפריט?
**ת:** לא בעריכה. אבל אפשר:
1. למחוק את המיפוי הישן (🗑️)
2. להוסיף מיפוי חדש (+ הוסף מיפוי) עם השם החדש

## טיפים

### 💡 תכנן מראש
לפני שמתחילים להזין עסקאות, כדאי להגדיר את כל המיפויים. אם צריך לשנות - עריכה קלה!

### 💡 קטגוריות אחידות
השתמש באותן קטגוריות לפריטים דומים. למשל:
- כל החנויות → "מזון"
- כל הרופאים → "טיפול רפואי"
- כל הנסיעות → "תחבורה"

### 💡 בדוק לפני שינוי
לפני ששולחים את העדכון, ודא שהקטגוריה החדשה נכונה.
