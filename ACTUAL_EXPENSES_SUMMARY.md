```
✅ תחנת דלק (כרטיס): -₪250        → יופיע בטבלה
❌ משכורת: +₪15,000                → לא יופיע
✅ רופא שיניים (מזומן): -₪420     → יופיע בטבלה
❌ צ'יק לחשמל: -₪800              → לא יופיע
```

#### הטבלה תראה:
```
💰 הוצאות ינואר
הוצאות בפועל (ללא תשלומי צ'יקים וללא הכנסות)

┌──────────────────┬──────────────┬────────┐
│ קטגוריה         │ סכום         │ אחוז   │
├──────────────────┼──────────────┼────────┤
│ מזון             │ ₪630.00      │ 48.5%  │
│ טיפול רפואי     │ ₪420.00      │ 32.3%  │
│ נסיעות          │ ₪250.00      │ 19.2%  │
├──────────────────┼──────────────┼────────┤
│ סה"כ הוצאות בפועל: │ ₪1,300.00 │ 100%   │
└──────────────────┴──────────────┴────────┘
```

#### הסבר:
- ✅ **₪1,300** = הוצאות שבאמת יצאו מהכיס החודש
- ❌ **לא כולל ₪2,000** של צ'יקים (ייפרעו בעתיד)
- ❌ **לא כולל ₪15,000** משכורת (זו הכנסה)

## 💡 למה זה שימושי?

### 1. 💳 תמונה אמיתית של ההוצאות
```
❌ לפני: "סה"כ -₪18,300 (כולל צ'יקים + הכנסות = מבלבל!)
✅ עכשיו: "הוצאות בפועל: ₪1,300" (ברור וממוקד!)
```

### 2. 🎯 ניהול תקציב
```
התקציב שלי: ₪3,000
הוצאות בפועל: ₪1,300
נותרו: ₪1,700 ✓

עכשיו אני יודע כמה נשאר לי באמת!
```

### 3. 📊 זיהוי דפוסי הוצאה
```
מזון: 48.5% ← זה הרבה! אולי אפשר לחסוך?
נסיעות: 19.2% ← סביר
טיפול רפואי: 32.3% ← הוצאה חד פעמית
```

### 4. 📈 השוואה בין חודשים
```
ינואר: ₪1,300
פברואר: ₪2,100 (עלה ב-₪800!)
מרץ: ₪1,500 (ירד בחזרה)

→ רואה מגמות ברורות
```

## 🔍 ההבדל מטבלאות אחרות

### בטאב "תצוגה חודשית" יש עכשיו 4 טבלאות:

#### 1️⃣ סיכום יתרות ותזרים חודשי
```
מה: יתרת פתיחה, הכנסות, הוצאות, יתרת סגירה
מטרה: לעקוב אחר מצב החשבון
```

#### 2️⃣ סיכום לפי קטגוריות
```
מה: כל התנועות (הכנסות + הוצאות + צ'יקים)
מטרה: תמונה מלאה של כל הקטגוריות
```

#### 3️⃣ הוצאות החודש 🆕
```
מה: רק הוצאות בפועל (ללא צ'יקים וללא הכנסות)
מטרה: כמה באמת הוצאתי מכיס החודש?
👉 זו הטבלה החדשה שהוספתי!
```

#### 4️⃣ תנועות החודש
```
מה: רשימה מפורטת של כל העסקאות
מטרה: לראות כל תנועה בנפרד
```

#### 5️⃣ תשלומי צ'יקים בחודש
```
מה: רק צ'יקים
מטרה: מעקב אחר התחייבויות עתידיות
```
