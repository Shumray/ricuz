# ✅ טבלת הוצאות בפועל - תצוגה חודשית

## תאריך: 31 ינואר 2026
## גרסה: 1.9.0

---

## 🎯 מה נוסף?

נוספה טבלה חדשה בטאב **"📅 תצוגה חודשית"** שמציגה את **הוצאות החודש בפועל**.

### מה מוצג בטבלה?
✅ **רק הוצאות** (לא הכנסות)  
✅ **ללא תשלומי צ'יקים** (רק הוצאות שנעשו במזומן/כרטיס)  
✅ **סיכום לפי קטגוריות** עם אחוזים  
✅ **סה"כ הוצאות בפועל** בתחתית הטבלה

## 🔍 מה לא כלול בטבלה?

### ❌ לא כלול:

#### 1. תשלומי צ'יקים
```
למה? כי צ'יקים הם התחייבות עתידית, לא הוצאה בפועל היום.

דוגמה:
- כתבת צ'יק ל-5 פברואר על סך ₪500
- בינואר: לא יופיע בטבלת "הוצאות ינואר"
- בפברואר: לא יופיע בטבלת "הוצאות פברואר" (כי זה צ'יק)
- יש טבלה נפרדת "🏦 תשלומי צ'יקים בחודש" בסוף העמוד
```

#### 2. הכנסות
```
למה? כי זו טבלת הוצאות, לא הכנסות.

דוגמה:
- משכורת: ₪15,000 → לא יופיע
- ביטוח לאומי: ₪3,500 → לא יופיע
- רק הוצאות ממשיות יופיעו
```

## 💡 למה זה שימושי?

### 1. 💳 הבנת הוצאות ממשיות
```
רוצה לדעת כמה באמת הוצאת החודש?
→ הטבלה הזו מראה לך את התמונה האמיתית
→ ללא צ'יקים שעוד לא נפרעו
→ רק מה שיצא מהכיס באמת
```

### 2. 📊 פילוח לפי קטגוריות
```
איפה הכסף הלך?
→ מזון: 45.5% (הכי הרבה!)
→ נסיעות: 24.7%
→ טיפול רפואי: 15.3%
→ עכשיו יודע איפה לחסוך
```

### 3. 🎯 ניהול תקציב
```
התקציב שלי: ₪3,000 לחודש
הוצאות בפועל: ₪2,750
→ נשארו ₪250 ✓
→ בתוך התקציב!
```

### 4. 📈 השוואה בין חודשים
```
ינואר: ₪2,750
פברואר: ₪3,200 (עלה!)
מרץ: ₪2,500 (ירד)
→ רואה מגמות
```
