# מדריך למשתמש - מערכת תקציב גיא שומרי

## איך להוסיף עסקה חדשה?

### שלב 1: בחירת חודש
בראש הטאב "📊 הזנת עסקאות" תמצא בורר חודש:
```
חודש: [פברואר ▼]  [+ הוסף עסקה]
```

1. לחץ על הבורר ובחר את החודש הרצוי
2. הטבלה מתחת תציג את כל העסקאות של החודש שבחרת
3. **החודש שבחרת יישמר** - גם אם תסגור את האתר, בפעם הבאה הוא יהיה ברירת המחדל!

### שלב 2: הוספת עסקה
1. לחץ על הכפתור **"+ הוסף עסקה"**
2. טופס יפתח עם השדות הבאים:
   - **פריט**: שם הפריט (למשל: "סופרסל", "רופא שיניים")
   - **סכום**: סכום בשקלים
   - **הערה**: הערה אופציונלית
   - **תשלום בצ'יק**: סמן אם זה תשלום בצ'יק

3. לחץ **"💾 שמור"**

**העסקה תתווסף אוטומטית לחודש שבחרת בבורר העליון!**

## דוגמה מעשית

### תרחיש: הזנת עסקאות לחודש פברואר

```
1. בחר "פברואר" בבורר החודש בראש הטאב
2. לחץ "הוסף עסקה"
3. הזן: פריט = "רמי לוי", סכום = 350
4. שמור → העסקה נוספה לפברואר ✓

5. לחץ שוב "הוסף עסקה" 
6. הזן: פריט = "משכורת", סכום = 15000
7. שמור → העסקה נוספה לפברואר ✓

8. רוצה להוסיף עסקה למרץ?
   → החלף את הבורר ל"מרץ" ← והמשך להוסיף עסקאות
```

## טיפים חשובים

### 💾 החודש הנבחר נשמר!
כשבוחרים חודש בבורר, המערכת זוכרת את הבחירה:
- ✅ גם אם תסגור את האתר
- ✅ גם אחרי שעות, ימים, או שבועות
- ✅ בפעם הבאה שתפתח את האתר - החודש שבחרת יהיה ברירת המחדל

### ⚠️ לא ניתן להוסיף עסקה כשבחרת "כל החודשים"
אם בורר החודש מוגדר ל-"כל החודשים", לחיצה על "הוסף עסקה" תציג הודעת שגיאה.
**פתרון**: בחר חודש ספציפי לפני הוספת עסקה.

### 📝 עריכת עסקה
כשעורכים עסקה קיימת, היא תישאר באותו חודש שבו נוצרה במקור.
זה מונע הזזה בטעות של עסקאות בין חודשים.

### 🔍 צפייה בעסקאות
- **חודש ספציפי**: הטבלה תציג רק עסקאות של אותו חודש
- **כל החודשים**: הטבלה תציג את כל העסקאות של השנה

### 🎯 תזרים עבודה מומלץ
```
1. התחל בחודש הנוכחי (ברירת מחדל)
2. הזן את כל העסקאות של החודש
3. עבור לחודש הבא
4. חזור וחזור...
```

## שאלות נפוצות

**ש: איך אני יודע לאיזה חודש העסקה תתווסף?**  
ת: תסתכל על הבורר בראש הטאב - החודש שמוצג שם הוא החודש שאליו תתווסף העסקה.

**ש: מה קורה אם אני רוצה להעביר עסקה לחודש אחר?**  
ת: עדיף למחוק את העסקה ולהוסיף אותה מחדש בחודש הנכון.

**ש: למה הוסר שדה החודש מהטופס?**  
ת: כדי לפשט את התהליך ולמנוע בלבול. עכשיו החודש נבחר במקום אחד בלבד ברור ונוח.
