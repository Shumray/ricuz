# ✅ פריטי שיק מיובאים - כעת גם בתצוגה החודשית!

## מה השתנה?

פריטי השיק המיובאים (פריטים עם שם `(שיק)`) מוצגים כעת **בשני מקומות**:

### 1️⃣ עמוד "קלט נתונים" 
✅ כבר היה קיים

### 2️⃣ עמוד "תצוגה חודשית" 
🆕 **חדש!** נוסף עכשיו

---

## 🎯 איך זה עובד?

### בעמוד "תצוגה חודשית":

1. ✅ בחר חודש מהתפריט הנפתח
2. ✅ הקונטיינר מופיע **רק אם יש פריטי שיק לחודש זה**
3. ✅ הכותרת דינמית: `🏦 פרטי צ׳יק הוצאות חודש <שם החודש>`
4. ✅ מוצגים כל פריטי השיק לחודש הנבחר
5. ✅ ניתן לערוך (✏️) ולמחוק (🗑️) ישירות מהעמוד

## 🔧 השינויים הטכניים

### 1. קוד JavaScript (budget.js)

#### הוספת קריאה ב-updateMonthlyView:
```javascript
updateMonthlyView() {
    const selectedMonth = parseInt(document.getElementById('monthSelect').value);
    this.updateBalanceSummary(selectedMonth);
    this.updateCategorySummary(selectedMonth);
    this.updateActualExpensesSummary(selectedMonth);
    this.updateMonthlyTransactions(selectedMonth);
    this.updateCheckPaymentsSummary(selectedMonth);
    this.updateImportedCheckItemsMonthly(selectedMonth); // ← חדש!
}
```

#### פונקציה חדשה:
```javascript
updateImportedCheckItemsMonthly(month) {
    // מסנן פריטי שיק לפי חודש ושנה נוכחית
    // מציג אותם עם כפתורי עריכה ומחיקה
    // מעדכן את הכותרת דינמית
    // מוסיף סיכום כולל
}
```

#### פונקציה למחיקה מהעמוד החודשי:

## ✨ תכונות

### ✅ תצוגה אוטומטית
- הקונטיינר מופיע רק אם יש פריטי שיק לחודש
- הכותרת משתנה דינמית לפי החודש הנבחר

### ✅ עריכה מלאה
- כפתור ✏️ לעריכת כל הפרטים
- חלון עריכה זהה לזה שבעמוד "קלט נתונים"
- עריכת: מספר צ'יק, מוטב, עבור, הערה

### ✅ מחיקה
- כפתור 🗑️ למחיקה
- אישור לפני המחיקה
- רענון אוטומטי של התצוגה

### ✅ סיכום
- סכום כולל של כל פריטי השיק
- מספר הפריטים

### ✅ סינכרון
- כל שינוי נשמר ב-localStorage
- השינויים מתעדכנים בשני העמודים

## 🧪 איך לבדוק?

### שלב 1: רענן את הדפדפן
לחץ `F5` או `Ctrl/Cmd + R`

### שלב 2: ייבא נתונים (אם עוד לא)
1. עבור ל**"קלט נתונים"**
2. בחר **פברואר 2026**
3. לחץ **"📄 ייבוא דף חשבון"**
4. בחר את הקובץ `test_check_item.csv`

### שלב 3: עבור לתצוגה חודשית
1. לחץ על הטאב **"תצוגה חודשית"**
2. בחר **פברואר 2026** בתפריט הנפתח
3. גלול למטה

### שלב 4: ראה את פריטי השיק
תראה את הקונטיינר:
```
🏦 פרטי צ׳יק הוצאות חודש פברואר
```
עם כל פריטי השיק של החודש

### שלב 5: נסה לערוך
1. לחץ על ✏️ ליד פריט שיק
2. ערוך את הפרטים
3. שמור
4. התצוגה תתעדכן אוטומטית

### שלב 6: נסה למחוק
1. לחץ על 🗑️ ליד פריט שיק
2. אשר את המחיקה
3. הפריט יימחק והתצוגה תתעדכן
