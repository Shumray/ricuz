## 🔧 שינוי טכני

### הקוד שהשתנה:

#### לפני:
```javascript
const sortedTransactions = [...currentYearTransactions].sort((a, b) => {
    if (a.month !== b.month) return b.month - a.month;  // מיון לפי חודש
    return b.id - a.id;  // אז לפי תאריך הוספה
});
```

#### אחרי:
```javascript
const sortedTransactions = [...currentYearTransactions].sort((a, b) => {
    // Sort alphabetically by item name (א-ב)
    return a.item.localeCompare(b.item, 'he');  // מיון אלפביתי בעברית
});
```

### הסבר:
- `localeCompare()` - משווה מחרוזות באופן תואם לשפה
- `'he'` - מציין שהמיון הוא בעברית
- מיון אלפביתי: א, ב, ג, ד... ת
