# עדכון אחרון - ברירת מחדל לשדה "עבור"

## השינוי שבוצע

פריטי שיק מיובאים (פריטים עם השם `(שיק)` בדיוק) יקבלו כעת ברירת מחדל **חכמה** לשדה "עבור":

### לפני:
```
עבור: שיק
```

### אחרי:
```
עבור: הוצאות חודש פברואר
```

## איך זה עובד?

כאשר מייבאים קובץ עם פריט `(שיק)`, המערכת:
1. ✅ קוראת את החודש מהשורה בקובץ
2. ✅ ממירה את מספר החודש לשם בעברית
3. ✅ יוצרת את הטקסט: `הוצאות חודש <שם החודש>`
4. ✅ שומרת את זה בשדה "עבור"
