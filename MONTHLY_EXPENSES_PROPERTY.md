# ✅ מערכת הוצאות חודשיות מבוססת Property

## גרסה: 2.0.0 | תאריך: 31 ינואר 2026

---

## 🎯 מה השתנה?

במקום להשתמש בקטגוריה נפרדת, כל מיפוי עכשיו כולל **property חדש**: `includeInMonthlyExpenses`

### מבנה ה-Mapping החדש:

```json
{
  "mappings": {
    "רמי לוי": {
      "category": "כלכלה",
      "includeInMonthlyExpenses": true    ← חדש!
    },
    "הראל": {
      "category": "מיסים",
      "includeInMonthlyExpenses": false   ← לא נכלל
    }
  }
}
```

## ⚙️ מה לא נכלל (כברירת מחדל):

✅ **הגדרת המיפויים הקיימים**:

### לא נכלל (❌):
- משכורת
- ביטוח לאומי
- מענק עבודה
- הראל
- מנורה
- פניקס
- עמלת בנק
- ע מפעולות ישיר
- מכבי

### נכלל (✅):
- כל השאר (סופרסל, רמי לוי, מגא, רופא, ביגוד, וכו')

## 🔧 איך להוסיף/לערוך מיפוי?

### הוספת מיפוי חדש:
1. טאב "🗺️ מיפוי קטגוריות"
2. לחץ "➕ הוסף מיפוי"
3. מלא:
   - **פריט**: שם הפריט
   - **קטגוריה**: בחר קטגוריה
   - **☑️ כלול בטבלת "הוצאות החודש"**: סמן/בטל סימון
4. לחץ "💾 שמור"

### עריכת מיפוי קיים:
1. לחץ ✏️ ליד המיפוי
2. שנה את הקטגוריה או את הסימון
3. לחץ "💾 עדכן"
